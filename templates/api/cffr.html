CFFR

URL Format:
http://data.nationalpriorities.org/api/<source>/list.<format>?...

Sources

There are four sources for CFFR data. They are

  • cffr
    Primary CFFR Data
    Allowed Keys: 'year', 'state_ansi', 'state_abbr', 'county_ansi', 'program_code'
    Example: http://data.nationalpriorities.org/api/cffr/list.json?state_ansi=01&county_ansi=001&year=2008
  • cffragency
    CFFR Agency Codes
    Allowed Keys: 'id', 'year', 'agency_code', 'agency_name'
    Example: http://data.nationalpriorities.org/api/cffragency/list.xml?year=2008
  • cffrgeo
    CFFR Geo Codes
    Allowed Keys: 'id', 'year', 'state_code', 'county_code', 'place_code', 'place_name', 'state_gu', 'type_gu', 'county_gu', 'place_gu', 'split_gu', 'congress_district'
    Example: http://data.nationalpriorities.org/api/cffrgeo/list.xml?state_code=40
  • cffrobjectcode
    CFFR Object Codes
    Allowed Keys: 'id', 'object_code', 'category'
    Example: http://data.nationalpriorities.org/api/cffrobjectcode/list.json
  • cffrprogram
    CFFR Programs
    Allowed Keys: 'year', 'program_code', 'program_name'
    Example: http://data.nationalpriorities.org/api/cffrprogram/list.xml?year=2008
